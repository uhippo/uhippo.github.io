Express(1)

路由的定义由如下结构组成：app.METHOD(PATH, HANDLER)。其中，app 是一个
express 实例；METHOD 是某个 HTTP 请求方式中的一个；PATH 是服务器端的路
径；HANDLER 是当路由匹配到时需要执行的函数。

app.all() 是一个特殊的路由方法，没有任何 HTTP 方法与其对应，它的作用是对于一个路径上的所有请求加载中间件。

使用字符串模式的路由路径示例:
// 匹配 /abe 和 /abcde
app.get(‘/ab(cd)?e’, function(req, res) {
res.send(‘ab(cd)?e’);
});

使用正则表达式的路由路径示例：
// 匹配 butterfly、dragonfly，不匹配 butterflyman、dragonfly man等
app.get(/.fly$/, function(req, res) {
res.send(‘/.fly$/‘);
});
